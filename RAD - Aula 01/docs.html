CalcAPI

API de cálculo aritmético — simples, rápida e documentada

Base URL: http://localhost:5000
Formato: application/json
Auth: Nenhuma
Operações disponíveis
+
soma
−
subtracao
×
multiplicacao
÷
divisao
BASE http://localhost:5000
Endpoints
POST /calcular Realiza o cálculo ▼
Request Body — application/json
CampoTipoDescrição
num1numberPrimeiro númeroobrigatório
num2numberSegundo númeroobrigatório
operacaostringsoma | subtracao | multiplicacao | divisaoobrigatório
Exemplo de requisição
{
  "num1": 15,
  "num2": 4,
  "operacao": "divisao"
}
Resposta 200 — Sucesso
{
  "sucesso": true,
  "operacao": "divisao",
  "expressao": "15 ÷ 4 = 3.75",
  "num1": 15,
  "num2": 4,
  "resultado": 3.75,
  "timestamp": "2026-03-07T14:00:00Z"
}
Códigos de resposta
200Cálculo realizado com sucesso
400Campos ausentes, tipo inválido ou operação desconhecida
422Divisão por zero
Exemplo curl
# Linux / macOS
curl -X POST http://localhost:5000/calcular \
  -H "Content-Type: application/json" \
  -d '{"num1": 15, "num2": 4, "operacao": "divisao"}'
